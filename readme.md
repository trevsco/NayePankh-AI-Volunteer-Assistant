# 🌿 NayePankh AI Volunteer Assistant

An AI-powered volunteer assistant built using Streamlit and Google Gemini API.

## Features

* 🤖 Gemini AI integration
* 🧠 Conversation memory
* 💡 Suggested questions
* 📥 Download chat history
* 🤝 Volunteer guidance
* 🎓 Internship information
* 🎨 Modern Streamlit UI

## Tech Stack

* Python
* Streamlit
* Google Gemini API
* python-dotenv

## Project Structure

```
nayepankh-ai-agent/
│
├── app.py
├── gemini_service.py
├── requirements.txt
├── README.md
├── screenshots/
├── .gitignore
└── .env
```

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
